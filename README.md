# Notes App

## Description

JavaScipt note-taking application.

### Deployment

This app is deployed to [Surge](https://hushed-voyage.surge.sh/).

### Built With

- [Webpack](https://webpack.js.org/) - Module bundler
- [Babel](https://babeljs.io/) - JavaScript compiler
- [uuid](https://www.npmjs.com/package/uuid) - Unique identifier creation
- [Moment](https://www.npmjs.com/package/moment) - Date parsing and formatting
